Potrebno imati:
- Python

Kako instalirati:
1. Preuzmite "main.py", "install.py", "requirements.txt"
2. Pokrenite "install.py" (nakon prvog pokretanja nije potrebno
ponovno pokretati)
3. Iz foldera "tools" izaberite alat koji vam je potreban te
preuzmite folder
4. Folder premjestite u isti folder u kojem je "main.py"

Pokretanje:
1. Pokreni main.py
